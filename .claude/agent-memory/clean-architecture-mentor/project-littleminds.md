Little Minds é uma plataforma para ensinar e guiar pais sobre desenvolvimento da inteligência emocional de seus filhos.

**Por que isso importa:** Este contexto orienta decisões de design - as entidades devem refletir o domínio de educação emocional infantil.

**Como aplicar:**
- ForumPost representa compartilhamento de vivências dos pais
- Article representa conteúdo especializado/educativo
- Nomenclatura deve ser acessível para pais, não técnica

**Funcionalidades principais:**
- Fórum comunitário para troca de experiências
- Artigos especializados
- Chat com IA para suporte
- Suporte para todas as faixas etárias
